
# Сервис транслитерации

Веб-приложение для транслитерации русского текста в латиницу. История запросов хранится в LevelDB.

## Возможности

- **Главная страница** — два поля ввода: набираете кириллицу, рядом мгновенно появляется латиница (без перезагрузки).
- **POST /api** — JSON API для транслитерации.
- **GET /history?n=N** — получение последних N результатов транслитерации.
- **LevelDB** — все запросы сохраняются в базу данных.

## Запуск

```bash
npm install
npm start
```

Сервер запустится на `http://localhost:3000`.

## API

### POST /api

Запрос:
```json
{"data": "апишка"}
```

Ответ:
```json
{"status": "success", "data": "apishka"}
```

### GET /history?n=5

Ответ:
```json
{"data": ["apishka", "voda"]}
```

## Технологии:

- Node.js, Express
- LevelDB (пакет `level`)
